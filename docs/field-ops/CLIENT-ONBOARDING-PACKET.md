# CLIENT ONBOARDING PACKET — Arizona Field Coverage

Welcome to Already Here LLC field coverage. This packet gets a new retainer
client fully operational within 30 days.

## Week 1 — Setup
- [ ] Signed MSA + Retainer Agreement + SLA on file
- [ ] Billing contact + PO requirements captured; first invoice issued
- [ ] Client points of contact recorded (primary, escalation, after-hours)
- [ ] Provider contacts delivered: dispatch@alreadyherellc.com + phone
- [ ] Site list with addresses, access/badging procedure per site
- [ ] Security/compliance requirements collected (badging, escorts,
      background checks, PPE, infection control for healthcare sites)
- [ ] Ticketing/integration preference: email, client portal, or
      client-system access for Provider

## Week 2 — Baseline
- [ ] Kickoff call: priorities, known problem sites, upcoming projects
- [ ] Initial site assessment scheduled for [N] primary site(s)
      (included in retainer hours)
- [ ] Asset inventory started at primary site(s)
- [ ] Photo documentation baseline captured

## Weeks 3–4 — Cadence
- [ ] First monthly utilization report delivered
- [ ] Standing maintenance windows agreed (if any)
- [ ] Escalation test: client submits a test S3 request end-to-end
- [ ] QBR schedule set (Business/Enterprise tiers)

## What Every Visit Includes
Work log entry, photo documentation, parts/serials recorded, and next-action
recommendation. Monthly you receive: hours used/remaining, visit list, SLA
performance, and flagged risks.

## How to Request Service
1. **Web:** www.alreadyherellc.com/dispatch (fastest routing)
2. **Email:** dispatch@alreadyherellc.com
Include: site, contact, severity (S1/S2/S3 per SLA), and scope notes.
