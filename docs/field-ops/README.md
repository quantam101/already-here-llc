# Arizona Field Operations Partner — Client Document Set

Sales and contracting documents for the Already Here LLC field-services retainer
program (`/arizona-field-coverage` on the website).

| Document | File | Use |
|---|---|---|
| Master Service Agreement | [MSA.md](MSA.md) | Governing terms, signed once per client |
| Retainer Agreement | [RETAINER-AGREEMENT.md](RETAINER-AGREEMENT.md) | Tier, hours, price — signed per retainer term |
| Service Level Agreement | [SLA.md](SLA.md) | Response times and remedies, attached to retainer |
| Statement of Work template | [SOW-TEMPLATE.md](SOW-TEMPLATE.md) | Per-project scope for professional services |
| Rate Card | [RATE-CARD.md](RATE-CARD.md) | Standalone and overage pricing |
| Client Onboarding Packet | [CLIENT-ONBOARDING-PACKET.md](CLIENT-ONBOARDING-PACKET.md) | First-30-days checklist for new retainer clients |
| Quarterly Business Review template | [QBR-TEMPLATE.md](QBR-TEMPLATE.md) | Quarterly utilization + value report |
| Assessment Report template | [ASSESSMENT-REPORT-TEMPLATE.md](ASSESSMENT-REPORT-TEMPLATE.md) | Deliverable format for all assessment services |
| Target Markets & Outreach | [TARGET-MARKETS-OUTREACH.md](TARGET-MARKETS-OUTREACH.md) | Prospect segments, sequences, tracking fields |

**Legal note:** MSA, Retainer Agreement, SLA, and SOW are working drafts written
for business review. Have an Arizona-licensed attorney review before first
client signature. Bracketed `[fields]` are per-client fill-ins.

Pricing source of truth: the retainer tiers and standalone rates on
`/arizona-field-coverage`. If pricing changes, update the page and
RATE-CARD.md together.
