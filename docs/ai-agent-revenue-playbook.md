# AI Web Agent Revenue Playbook

## Offer

Already Here LLC sells productized AI website agents that capture leads, qualify quote requests, route owner alerts, and create structured lead records.

Primary product: **Already Here AI Web Agent**

## Pricing

| Package | Setup | Monthly | Buyer |
|---|---:|---:|---|
| Launch Agent | $997 | $197/mo | One-site service business |
| Growth Agent | $1,997 | $397/mo | Service business with repeat quote volume |
| Network Agent | $4,500+ | $997+/mo | Multi-location, dispatch, or technician network |

## Target buyers

Start with buyers where one closed lead can pay for setup:

- IT support and MSPs
- Hauling and delivery
- Roofing and home services
- Clinic intake
- Restaurant and retail service
- Real estate investor intake
- Field technician networks

## Positioning

Do not sell abstract AI. Sell:

- Fewer missed leads
- Faster quote response
- Better intake data
- Owner alerts
- Follow-up readiness
- Lead records that can be exported, reviewed, and optimized

## Qualification questions

1. How many website leads, calls, or quote requests do you get per month?
2. How many are missed, delayed, or poorly qualified?
3. What is one closed customer worth?
4. Who should receive the alert?
5. What fields must the agent capture before a lead is useful?
6. What should never be automated without approval?
7. What website platform are you using?
8. Do you want one location, multiple locations, or territory routing?

## Sales math

The simplest close is:

> If one closed lead is worth more than the setup fee, the agent only needs to save or create one extra job to justify the install.

For Already Here LLC, the first goal is 5 Launch Agent clients:

- Setup revenue: 5 x $997 = $4,985
- Monthly recurring revenue: 5 x $197 = $985/mo

Then convert 2 of 5 into Growth Agent:

- Added setup revenue: 2 x $1,000 upgrade gap = $2,000
- Added monthly revenue: 2 x $200/mo upgrade gap = $400/mo

## Call close

Use this sequence:

1. Confirm lead value.
2. Confirm missed-lead problem.
3. Show the live AI Agent Demo page.
4. Explain setup plus monthly management.
5. Offer Launch Agent unless routing complexity requires Growth Agent.
6. Ask for website access and intake details.
7. Send invoice or payment link after approval.

## Review loop

Weekly:

- Check site health.
- Check `/api/ai-agent-lead` health.
- Review new lead records.
- Identify objections and missing questions.
- Improve widget copy.
- Add one target vertical page or sales angle.

Monthly:

- Review close rate.
- Raise price when demand exceeds fulfillment time.
- Package repeated configurations as reusable vertical templates.
