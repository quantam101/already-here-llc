# Decisions

## D-001 - GitHub is the long-term source of truth

Status: accepted

Durable engineering work is governed through GitHub.

## D-002 - Vercel is the default production runtime

Status: accepted

The live site remains on Vercel unless disproven.

## D-003 - www is the canonical production host

Status: accepted

www.alreadyherellc.com is primary. alreadyherellc.com redirects to www.

## D-004 - Preserve working systems first

Status: accepted

Working pages, routes, forms, and deploy paths are preserved before restructuring.

## D-005 - Commercial framing is field execution, not generic IT help

Status: accepted

Customer-facing content should read like a controlled field execution system.

## D-006 - Dispatch is the primary controlled conversion path

Status: accepted

The main action is dispatch request intake, not vague general inquiry.

## D-007 - Governance docs are required repo infrastructure

Status: accepted

Required docs:

- README.md
- ARCHITECTURE.md
- DECISIONS.md
- RUNBOOK.md
- docs/BUILD_OPERATING_STANDARD.md
- .env.example

## D-008 - CI, smoke testing, and observability are mandatory next layers

Status: accepted
