# Daily Command Summary Master Prompt

Use this prompt to generate the Already Here LLC Daily Command Summary from connected operating sources.

```text
Search connected Gmail and GitHub for Already Here LLC operating signals. Produce a Daily Command Summary covering AH/Hot, AH/Needs-Action, Field Nation, WorkMarket, dispatch intake, procurement notices, billing/payment items, website/GitHub health, drone/UAS opportunities, V2D Portal / Vets to Drones emails, free veteran drone training, drone certifications, FAA Part 107 pathways, and paid drone work leads.

For V2D/drone items, search Gmail for from:portal@vetstodrones.org, from:(vetstodrones.org), subject:(V2D), subject:(SITREP), "UAS Industry Intelligence", "Vets to Drones", "free training", "certification", "Part 107", "FAA", "drone job", "UAS job", "mapping", "inspection", "pilot", "contract", "veteran", and "scholarship".

Return only actionable or materially changed items with source, date, deadline, eligibility, cost/free status, apply/contact link if available, grade, risk flags, and exact next action. Include the top 3 ranked actions.

Do not send emails, delete emails, archive emails, submit forms, enroll, apply, move money, change credentials, or modify production files without approval.
```

## Ranking standard

Grade each item by immediate business impact, deadline pressure, revenue probability, operational risk, and cost/risk exposure.

Suggested grades:

- A: immediate action, direct revenue/funding/security impact, or hard deadline
- B: meaningful opportunity or operational risk, not yet urgent
- C: monitor only or low direct impact
- D: suspicious, low value, or likely negative action

## Required item fields

- Source
- Date
- Subject/title
- Material change
- Deadline
- Eligibility
- Cost/free status
- Apply/contact link if available
- Grade
- Risk flags
- Exact next action
