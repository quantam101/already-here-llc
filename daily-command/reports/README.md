# Daily Command Reports

This directory is intentionally public-safe.

Do not commit raw daily reports containing:

- Gmail snippets or full email bodies
- personal data
- vendor/customer private details
- invoices, payment data, balances, disputes, or account notices
- credentials, tokens, API keys, secrets, or links containing tracking/auth data
- production incident details that expose vulnerabilities

Use this folder only for sanitized examples, redacted summaries, templates, and non-sensitive operating documentation.

Recommended private path once a private repo exists:

```text
quantam101/daily-command-private/reports/YYYY/YYYY-MM-DD-daily-command-summary.md
```

Required private report naming convention:

```text
YYYY-MM-DD-daily-command-summary.md
```
